# **About This Folder**

Although Labled JScriptSamples it includes JavaScripted and VBS HTML/HTA sample files too

## HTML

HTML based PDF.html is a demonstration of a "Self embedded PDF" as text with an image and supporting HTML iFrame thus is not subject to CORS.


## HTA

Disposition.hta is a tool to download remote URLs to see how it works copy this file of coma seperated local and remote references. Place as "disposition.txt" alongside Disposition.hta then double click the downloaded HTA file.  
Disposition.txt
```
example.html,https://www.example.com
example.pdf,https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf
```
